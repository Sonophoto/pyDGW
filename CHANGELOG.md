## [1.0.0] - 2026-02-05

### Overview
First stable release of pyDGW - a Python teaching library for understanding and implementing finite state machines using directed graphs.

### Added
- Complete implementation of DGWalker class for graph traversal
- DGW_data base class for state management
- Comprehensive test suite with unittest framework
- Multiple working examples:
  - SimpleExample.py - Basic state machine usage
  - CounterExample.py - Counter with state-based control
  - FlipFlopExample.py - Binary state toggling
  - TurnstyleExample.py - Interactive 5-state machine
  - CoinOpExample.py - Complex 7-state real-world example
- GitHub Actions CI/CD pipeline testing Python 3.8-3.11
- Standard Python packaging with setup.py for pip installation
- Comprehensive documentation in README.md

### Changed
- Updated to modern Pythonic coding style (February 2026 refresh)
- Improved code clarity and maintainability
- Enhanced documentation with clearer examples
- Updated copyright dates to 2016-2026

### Documentation
- Complete API documentation in README.md
- Detailed usage instructions with step-by-step guide
- Multiple tutorial examples with increasing complexity
- Troubleshooting guide with debug flag explanations
- Links to educational resources on state machines

### Technical Details
